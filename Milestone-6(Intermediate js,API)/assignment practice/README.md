*1) var, let, const*

* var = function/global scope, hoisted, can re-declare & update.
* let/const = block scope; let can update, const can’t reassign.


*2) map(), forEach(), filter()*

* forEach() → runs on each element, returns nothing.
* map() → returns new array of transformed values; filter() → returns new array of matching values.


*3) Arrow functions (ES6)*

* Shorter syntax for functions with lexical this.
* Cannot be used as constructors (new not allowed).


*4) How does destructuring assignment work in ES6?*

* Lets you unpack values from arrays/objects into variables.
* Supports defaults and renaming.


*5) Explain template literals in ES6. How are they different from string concatenation?*

* Strings with backticks `` ` ``, support ${} interpolation.
* Easier than + concatenation and support multiline.
